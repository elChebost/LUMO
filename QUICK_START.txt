   ❓ El script no arranca
      ➜ chmod +x start.sh

   ❓ Puerto ya en uso (3000 o 5173)
      ➜ lsof -ti:3000 | xargs kill -9
      ➜ lsof -ti:5173 | xargs kill -9

   ❓ npm-run-all no encontrado
      ➜ npm install

   ❓ El frontend no conecta con el backend
      ➜ Verificar backend en http://localhost:3000/api/students
      ➜ Ver frontend/src/config/api.js debe apuntar a puerto 3000
